06 Scope — Stage 1 vs Stage 2
Building Now
Stage 1
✅
General
Catalog visibility, default currency, platform name
✅
Access / Launch
Platform mode, password gate, invite system, Go Live flow
✅
Auction
Anti-snipe window, block duration, stagger, direct purchase, reminders
✅
Change History
Full audit log with actor, old/new values, timestamps
Deferred — Built When Needed
Stage 2
⏸️
Payments
Stripe / PayPal toggle, accepted methods, fee visibility
⏸️
Email Notifications
Per-email-type toggles (welcome, outbid, shipping, feedback)
⏸️
Integrations
Discogs sync, Sentry error tracking, RudderStack analytics
⏸️
Feature Flags
Granular per-feature toggles for gradual rollout
Note: Stage 2 is documented and designed — tabs will be present in the UI but marked "Coming Soon". Each will be activated when the operational need arises post-launch. No rework of Stage 1 required.

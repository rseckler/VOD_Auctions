
# Purchase
- Im User Profil muss ein Bereich sein, in dem er seine früheren Bestellungen sehen kann
- Shipping Back-End:
	- Die Auswahl der Versandkosten muss je Land möglich sein
	- Die Shipping-Konfiguration muss um diese Möglichkeit erweitert werden
	- Recherchiere Best Practice Beispiele, wie z.b. bei Shopify oder ähnliche Anbieter, wie das Shipping Thema Marktüblich gelöst werden sollte
	- Vor allem die Frage nach dem Gewicht
	- Wir benötigen die Möglichkeit einen Versandanbieter mit den einzelnen Konditionen zu hinterlegen
	- Die Funktion muss einfach und modern zu bedienen sein
- Shipping für End-User:
	- Für den User muss es Marktüblich einfach sein
	- Je land individuell für die Kosten
